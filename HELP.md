# Getting Started

Demo to show the usage of the OPEN API in Spring Boot.

Please see the [Turtorial](https://www.baeldung.com/spring-rest-openapi-documentation).

After the Service has been started, check out the following URLS:
* [Open API Spec in JSON](http://localhost:8080/v3/api-docs)
* [Open API Spec in YAML](http://localhost:8080/v3/api-docs.yaml)
* [Open API UI](http://localhost:8080/swagger-ui)
